# FIS Shaft Intelligence Tool PWA

GitHub Pages-ready static PWA rebuilt from `Golf_Shaft_Tool_v8.1.xlsm`.

## Key changes
- Latest ShaftDB and Head_DB data exported into `data.js` and `/data/shaft_data.json`.
- Taper iron set rows remain in the data but are grouped for recommendations so AXIOM/taper sets do not appear once per iron club.
- `/icons` and service worker cache updated for GitHub Pages static hosting.

Upload these files to the root of a GitHub Pages repo, or into a subfolder if using a multi-PWA repo.
